# A股行情终端 Pro

A股 K 线 + 实时报价 + 多指标共振选股工具，纯前端，无需后端，直接打开 HTML 文件即可使用。

## 文件说明

| 文件 | 说明 |
|---|---|
| `stock-pro-screener-v4.html` | 桌面端主页面 |
| `stock-mobile.html` | 移动端主页面（手机优化布局） |
| `app.js` | 桌面端逻辑（指标计算 + 图表 + 扫描） |
| `app-mobile.js` | 移动端逻辑（完整独立） |
| `style.css` | 桌面端样式 |

## 功能

- **K 线图**：前复权日 K + MA5/10/20/60 + BOLL 布林带
- **MACD 副图**：DIF / DEA / 柱状图
- **自动分析**：8 项指标综合评分（MA / MACD / RSI / KDJ / MFI / OBV / 量能），ATR 波动区间模型
- **高级指标**：枢轴点 R1/S1/R2/S2、%B、BIAS、52 周百分位、缺口监测、相对强弱
- **多指标共振选股**：评分 ≥ 5/8 · 15 日上升趋势（R² ≥ 0.4）· RSI < 75 · KDJ-K < 80 · 偏离 MA20 < 15%
- **KDJ 金叉筛选**：仅返回当前 K 仍 > D 的有效金叉，支持超卖区 / J 上拐过滤
- **KDJ + MACD 双线共振**：两个金叉均当前有效，支持零轴 / 低位过滤
- **查看历史**：最近 30 条浏览记录，localStorage 持久化
- **移动端**：底部弹出式面板，无图表，轻量快速

## 使用方式

直接用浏览器打开对应 HTML 文件即可，数据来自腾讯股票接口（需能访问 `gtimg.cn`）。

> 排除 ST / 退市 / 当日涨幅 ≥ 10%；预测为 ATR 波动区间模型，非价格预测，不构成投资建议。
